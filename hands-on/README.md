# Hands-on projects

* [httpie-lite](./httpie-lite/README.md), minimal async HTTP client CLI to demo request/response handling, pretty printing, and basic POST/GET usage.
* [merkle-tree](./merkle-tree/README.md), simple Merkle Tree implementation with root computation, proof generation and verification.
* [rust-codes-flow](./rust-codes-flow/README.md), a fast CLI tool that converts any public Rust GitHub repository into a clean JSONL dataset optimized for training AI coding agents.
